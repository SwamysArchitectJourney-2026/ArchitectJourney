**Essential technical areas** you must be prepared to discuss in the Technical Round.

## ‚≠ê 3.2 Development Expertise

You don't need to code, but you must show:

* Strong understanding of C#, .NET 8, EF Core, API development
* How to review code
* How to guide developers technically
* How to debug issues

> A developer says an API is slow. How will you troubleshoot?

**Expected Answer:**
* Check logs ‚Üí profiling ‚Üí DB queries ‚Üí caching ‚Üí async ‚Üí bulk operations
