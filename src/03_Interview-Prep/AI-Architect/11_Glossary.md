# AI Architect Interview Glossary

> Fast reference for acronyms and terms used across this folder.

## Core Interview Acronyms

- **MTTR (Mean Time To Recovery)** – Average time to restore service after an incident.
- **ROI (Return on Investment)** – Benefit or value gained relative to cost.
- **STAR (Situation, Task, Action, Result)** – Framework for structuring behavioural answers.
- **RBAC (Role-Based Access Control)** – Authorization model based on user roles.
- **APIM (API Management)** – Managed API gateway used to expose, secure, and monitor APIs.
- **WAF (Web Application Firewall)** – Firewall that protects web apps from common web attacks.
- **CI (Continuous Integration)** – Practice of frequently integrating and validating code changes.
- **CD (Continuous Delivery / Deployment)** – Automating release of changes into test and production.
- **HPA (Horizontal Pod Autoscaler)** – Kubernetes controller that scales pods based on metrics.
- **KEDA (Kubernetes Event-Driven Autoscaling)** – Kubernetes component for event/queue-based scaling.
- **SKU (Stock Keeping Unit)** – Distinct pricing/feature tier for a service or product.
- **AKS (Azure Kubernetes Service)** – Managed Kubernetes service on Azure.
- **GPU (Graphics Processing Unit)** – Parallel compute accelerator, often for ML/AI workloads.
- **CPU (Central Processing Unit)** – General-purpose processor for application and control logic.
- **DNS (Domain Name System)** – Service that resolves human-readable names to IP addresses.
- **VNet (Virtual Network)** – Isolated, software-defined network boundary in the cloud.
- **QPS (Queries Per Second)** – Throughput metric for how many requests a system handles per second.
- **P95 (95th Percentile Latency)** – Latency below which 95% of requests complete.
- **P99 (99th Percentile Latency)** – Latency below which 99% of requests complete.
- **RAG (Retrieval-Augmented Generation)** – Pattern where an LLM uses retrieved context at query time.
- **DLP (Data Loss Prevention)** – Controls to prevent sensitive data exfiltration or misuse.
- **HNSW (Hierarchical Navigable Small World)** – Graph-based ANN index for fast vector search.
- **IVF (Inverted File Index)** – Vector index that clusters embeddings into coarse cells for search.
- **KNN (k-Nearest Neighbours)** – Algorithm that finds the k closest points in a vector space.
- **gRPC (gRPC Remote Procedure Calls)** – High-performance, binary RPC framework over HTTP/2.
- **REST (Representational State Transfer)** – Style for HTTP APIs using resources and standard verbs.
- **LLM (Large Language Model)** – Transformer-based model trained on large text corpora.
- **PII (Personally Identifiable Information)** – Data that can identify an individual.
- **OPA (Open Policy Agent)** – Policy engine used to enforce authorization and compliance rules.

## Additional Useful Terms

- **GitOps** – Operating model where desired state is stored in Git and reconciled to clusters.
- **MLOps** – Practices for managing ML lifecycle (training, deployment, monitoring).
- **SLO (Service Level Objective)** – Target for service reliability or performance.
- **SLA (Service Level Agreement)** – Contracted promise of availability or performance.
- **SRE (Site Reliability Engineering)** – Discipline focused on reliability using software engineering.
- **TTL (Time To Live)** – Expiry duration for cache or stored data.
- **KPI (Key Performance Indicator)** – Metric that tracks success against a business goal.
